<h1>Mood Tracker</h1>

<p>A simple Python OOP project to record daily moods with a note and date.</p>

<h2>Features</h2>
<ul>
  <li>Add a mood entry</li>
  <li>Store mood, note, and date</li>
  <li>View mood history</li>
  <li>Show mood summary</li>
  <li>Find the most frequent mood</li>
  <li>Handle invalid input safely</li>
</ul>

<h2>OOP Concepts Used</h2>
<ul>
  <li>Classes and objects</li>
  <li>Encapsulation</li>
  <li>Methods</li>
  <li>Input validation</li>
</ul>

<h2>Tech Used</h2>
<ul>
  <li>Python</li>
  <li>Object-Oriented Programming (OOP)</li>
</ul>
